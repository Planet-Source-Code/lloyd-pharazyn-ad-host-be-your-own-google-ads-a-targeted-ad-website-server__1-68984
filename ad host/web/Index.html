Hi and welcome to Ad Host, this program was made in Microsoft Visual Basic 6.0 Ent. and is a quick and VERY simple targeted ad hosting server
To add the image to the site add this in the html
<img src="http://127.0.0.1/ad=0001?1">

how it works
0001 = this is the account id which is used for billing or crediting and each time an ad is loaded it updates their account
1 = this is the ad category and you can have anything not just the number
